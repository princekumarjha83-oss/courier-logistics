# Courier Logistics Website

A modern, responsive courier logistics website built with HTML, CSS, and JavaScript.

## 🚀 Live Demo

**GitHub Pages URL:** https://princekumarjha83-oss.github.io/courier-logistics/

## 📋 Features

- 🏠 Modern landing page
- 📦 Package tracking system
- 🚚 Services showcase
- 📞 Contact form
- ❓ FAQ section
- 🔐 User authentication
- 📱 Fully responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3 with modern design
- Vanilla JavaScript
- Responsive UI/UX

## 🌐 Deployment

This project is deployed using GitHub Pages. To enable GitHub Pages:

1. Go to repository settings
2. Scroll to "Pages" section
3. Select "GitHub Actions" as source
4. Save settings
